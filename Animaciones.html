animation-name & animation-duration

Toda animación necesita como mínimo dos propiedades: Por un lado tenemos animation-name en la que asociamos el elemento que queremos animar con la propiedad que genera la animación, por otro, tenemos animation-duration que dertemina cuántos segundos durará dicha animación. Veamos como ejemplo el siguiente código:

                    #luzroja {
                        width: 100px;
                        height: 100px;
                        background-color: rgb(255, 0, 0);
                        border-radius: 100%;
                        animation-name: luz_roja;
                        animation-duration: 15s;
                    }
                    
                    @keyframes luz_roja {
                        15% {background-color: rgb(255, 0, 0);} 
                        20% {background-color: rgba(194, 0, 0, 0.829);}
                        60% {background-color: rgba(194, 0, 0, 0.829);}
                        65% {background-color: rgb(255, 0, 0);}
                        100% {background-color: rgb(255, 0, 0);}
                    }

Como se puede apreciar en el código de arriba tenemos el conjunto de reglas que hacen referencia al id #luzroja, dentro de esta regla definimos las características del objeto a animar (color, forma, tamaño, etc) del que partirá la animación y, además, hacemos uso de las dos propiedades ya mencionadas: animation-name y animation-duration. La primera asocia el elemento #luzroja a la regla @keyframes luz_roja que será la regla que ejecute la animación, la segunda determina la duración de la animación, en este caso, será de 15 segundos.

Una vez hemos fijado el tiempo de la animación (15s) podemos determinar cuando se realizarán cambios dentro de la misma con porcentajes. De forma que, usando el ejemplo de arriba, podemos apreciar que la luz roja parte de un estado inical en la regla #luzroja usando como color de fondo "rgb(255, 0, 0)", mientras que, dentro de la regla "@keyframes luz_roja" determinamos con porcentajes en qué momento de esos 15 segundos se van a producir cambios en la animación (en este ejemplo, los cambios solo serán de variación de color).

animation-delay

Antes de explicar la nueva propiedad, en este animación he añadido que el elemento se pueda desplazar en el eje x, en la regla que activa la animación está subrayada la característica que permite este desplazamiento. Lo único a tener en cuenta es que el desplazamiento cuenta la distancia en píxeles desde el margen determinado, en este caso, nos desplazamos 200 y 400px desde el margen izquierdo.

En esta animación vamos a añadir una propiedad nueva, animation-delay, con ella, vamos a hacer que la animación se active x segundos después de haberse cargado la página, en concreto, se activará 1s más tarde. Este es el código:

                    #anim2 {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        animation-name: animation2_3;
                        animation-duration: 4s;
                        animation-delay: 1s;
                    }
                    
                    @keyframes animation2_3 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:200px; top:0px;}
                        50%  {background-color:green; left:400px; top:0px;}
                        75%  {background-color:blue; left:200px; top:0px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }

En negrita resalto la nueva propiedad, esta propiedad que añadimos dentro de la regla que referencia al id de nuestro elemento, será la propiedad que determine cuanto tiempo tardará en activarse la animación tras la carga completa de la página.

animation-iteration-count

Esta propiedad, como se peude deducir de su nombre, añade la característica de repetir una animación ya creada n veces. Para este ejemplo, vamos a repetirla 3 veces. Dejo por aquí el código con la nueva propiedad en negrita: 

                    #anim3 {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        animation-name: animation2_3;
                        animation-duration: 4s;
                        animation-iteration-count: 3;
                    }
                    
                    @keyframes animation2_3 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:200px; top:0px;}
                        50%  {background-color:green; left:400px; top:0px;}
                        75%  {background-color:blue; left:200px; top:0px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }

Como dato de interés, podemos repetir de forma indefinida una animación asignando el valor infinite a la propiedad animation-iteration-count.

animation-direction

Esta propiedad determina la dirección o sentido con el que se va a reproducir la animación. Es una propiedad especial, ya que su comportamiento variará en función del valor que se le asigne. Es una propiedad a la que se le puede asignar cuatro valores, a saber: normal, reverse, alternate y alternate-reverse.

normal:

Con el valor normal la animación realiza el recorrido asignado, es decir, se realizará la animación por defecto como si no se hubiese empleado esta propiedad. A continuación el código de la animación: 

                    #anim4_normal {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        animation-name: animation4;
                        animation-duration: 4s;
                        animation-direction: normal;
                    }
                    
                    @keyframes animation4 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:100px; top:0px;}
                        50%  {background-color:green; left:100px; top:100px;}
                        75%  {background-color:blue; left:0px; top:100px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }
normal





reverse:

Si usamos el valor reverse la animación se ejecutará en sentido opuesto al señalado. En el caso anterior el cuadrado rotaba en sentido de las agujas del reloj pero, en este, rota en sentido opuesto.

                    #anim4_reverse {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        text-align: center;
                        animation-name: animation4;
                        animation-duration: 4s;
                        animation-direction: reverse;
                    }
                    
                    @keyframes animation4 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:100px; top:0px;}
                        50%  {background-color:green; left:100px; top:100px;}
                        75%  {background-color:blue; left:0px; top:100px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }

alternate:

El valor alternate ejecuta la animación tanto en sentido normal como inverso. Primeramente ejecutará la versión en el sentido normal y, posteriormente, la versión invertida. Para ello es imprescindible emplear la propiedad anterior animation-iteration-count para que haga, como mínimo, las dos vueltas.

                    #anim4_alternate {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        text-align: center;
                        animation-name: animation4;
                        animation-duration: 4s;
                        animation-iteration-count: 2;
                        animation-direction: alternate;
                    }
                    @keyframes animation4 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:100px; top:0px;}
                        50%  {background-color:green; left:100px; top:100px;}
                        75%  {background-color:blue; left:0px; top:100px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }
alternate





alternate-reverse:

Finalmente alternate-reverse que, como se puede deducir, es un alternate que, en vez de empezar la ejecución predefinida de la animación empieza por el sentido inverso y luego ejecuta el sentido normal. De igual modo, necesitaremos hacer uso de animation-iteration-count para hacer como mínimo las dos vueltas.

                    #anim4_alternate {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        text-align: center;
                        animation-name: animation4;
                        animation-duration: 4s;
                        animation-iteration-count: 2;
                        animation-direction: alternate-reverse;
                    }
                    @keyframes animation4 {
                        10%  {background-color:orange; left:0px; top:0px;}
                        25%  {background-color:yellow; left:100px; top:0px;}
                        50%  {background-color:green; left:100px; top:100px;}
                        75%  {background-color:blue; left:0px; top:100px;}
                        100% {background-color:blueviolet; left:0px; top:0px;}
                    }

animation-timing-function

Esta propiedad describe la curva de velocidad que tomará la animación, es decir, que en vez de mantener una velocidad constante, se puede establecer que en determinados momentos acelere o decelere según esté prefijado para cada valor de la propiedad. Éstos son los valores que puede tomar:

linear:

Con el valor linear la animación adquiere la velocidad por defecto, como si no se emplease esta propiedad.

ease:

Con el valor ease la animación arranca con un inicio ligeramente lento que se acelera progresivamente y, conforme se aproxima al final, decelera nuevamente.

ease-in:

Con ease-in la animación comienza con un inicio muy lento y, cercano al final, incrementa al máximo la velocidad.

ease-out:

Al emplear ease-out la animación comienza a la máxima velocidad y, cercano al final, la disminuye al mínimo.

ease-in-out:

Finalmente, con ease-in-out la animación comienza muy lentamente, acelera al máximo, alcanzando su máxima velocidad a medio recorrido, y de nuevo disminuye la velocidad al mínimo aproximándose al final.

Aquí una muestra de código del ejemplo ease-in:

                    #anim5_3 {
                        width: 100px;
                        height: 100px;
                        border-radius: 100%;
                        position: relative;
                        background-color: red;
                        text-align: center;
                        animation-name: animation5;
                        animation-duration: 6s;
                        animation-timing-function: ease-in;
                    }
                    @keyframes animation5 {
                        10%  {background-color:orange;}
                        25%  {background-color:yellow;}
                        50%  {background-color:yellow;}
                        75%  {background-color:orange;}
                        100% {background-color:red;}
                        from {left: 0px;}
                        to {left: 300px;}
                    }

Además de la propiedad aquí vista, señalo en negrita que también se le puede dar una posición de salida y de finalización a una animación de dicha manera.
